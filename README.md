# mini_ai_agent
---

### Python 3.10+ recommended
### Installation
```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### Run
```bash
python main.py
```
### Example output
```bash

hello

What is 12.5% of 243
```
